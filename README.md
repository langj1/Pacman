#Pacman

Tavoitteena on rakentaa perinteinen pacman peli, eli peli jossa liikutaan
kapeita käytäviä pitkin sokkelossa. Tavoitteena on kerätä kaikki pisteet
ilman, että osuu monstereihin. Pelaajalla on 3 elämää.
